# 🌟 SkillUp Connect

A Full-Stack web application designed to help NGOs manage and publish skill-building workshops for local youth.

## 🚀 Tech Stack
* **Frontend:** React.js, CSS3
* **Backend:** Node.js, Express.js
* **Database:** MySQL

## ✨ Features
* **Create:** NGOs can easily publish new workshops using a dynamic React form.
* **Read:** Users can view a real-time feed of all upcoming events.
* **Delete:** Instantly remove outdated or canceled workshops.
* **UI/UX:** Fully responsive, modern Dark Mode interface.

## 🛠️ How to Run Locally
1. Clone the repository.
2. Run `npm install` in both the frontend and backend folders.
3. Start your MySQL server and import the database.
4. Run `node server.js` in the backend.
5. Run `npm start` in the frontend.
